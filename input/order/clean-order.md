# Clean Order

Вставьте сюда очищенный текст приказа в формате Markdown.

Что желательно удалить заранее:
- лишние абзацы;
- следы старых диалогов;
- повторяющиеся команды;
- промежуточные пояснения, которые не должны попадать в рабочий шаблон.

Что желательно сохранить:
- итоговые обязательные инструкции;
- точный порядок действий;
- жесткие требования к формату вывода;
- все фразы, которые руководство считает обязательными.
