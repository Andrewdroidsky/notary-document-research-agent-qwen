# Part 02 Source Cascade

Каскад фиксирует приоритет слоев источников для URL1/URL2 и верификации.

## 1. URL1 official anchor

Домены: publication.pravo.gov.ru, pravo.gov.ru
Применение: Официальная публикация или официальный правовой портал как якорь официальности.

## 2. Official topical layer

Домены: minjust.gov.ru, notariat.ru, government.ru, kremlin.ru
Применение: Профильные официальные источники: Минюст, ФНП, Правительство, Президент.

## 3. Readable URL2 candidates

Домены: consultant.ru, garant.ru, docs.cntd.ru
Применение: Читаемые источники для VERIFIED URL2 после сверки реквизитов и заголовка страницы.
