Структура Оглавления должна быть такой: наименование основной темы м ниже - подтемы. Если Оглавление содержит несколько подтем, тогда каждой подтеме присваивается трехзначная цифра.
Например:

15.11.2. Наследование недвижимого имущества, в том числе земельных участков.
Если же основная тема имеет также несколько подтем, но которые LLM соединила в одну подтему, тогда мы просто делаем одну подтему,с присвоением двузначной цифры, например:
